## How to install this mod:

1. Prerequisites: Ensure you have the Darktide Mod Loader (DML) and Darktide Mod Framework (DMF) installed in your game folder.

2. Create Folder: Go to your Warhammer 40,000 DARKTIDE/mods/ directory and create a folder named AbilityTracker.

3. Place Files: Put the four files generated above inside that folder. Note that the .lua files belong in a subfolder: AbilityTracker/scripts/mods/AbilityTracker/.

4. Load Order: Open your mod_load_order.txt file (in the mods folder) and add AbilityTracker to a new line at the bottom.

5. Run Patcher: Run toggle_darktide_mods.bat in your game root folder to apply the changes.
